### RealityKit CardFlip

Example code for Apple's new AR framework RealityKit, demonstrating the following things:

- The Entity Component System
  - Inheritance/Protocol Oriented
  - AnchoringComponent (with Targets)
  - ModelComponent
  - CollisionComponent
- HitTests
- Animations
- AR Coaching
